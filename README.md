## 📌 소개

이 프로젝트는 주문, 상품, 고객, 통계를 관리하는 **자바 기반 애플리케이션**입니다.  
Swing GUI를 통해 사용자 친화적인 화면과 Oracle DB를 활용한 강력한 데이터 처리를 제공합니다.
